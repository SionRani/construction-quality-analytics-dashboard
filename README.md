# construction-quality-analytics-dashboard
Interactive Power BI dashboard for construction quality analytics, featuring aggregate, mechanical, and volumetric test analysis with KPIs, trends, and material performance insights. Inspired by QA/QC practices observed during my L&amp;T internship.
Construction Quality Analytics Dashboard
Project Overview

Developed an interactive Power BI dashboard to analyze construction quality using aggregate, mechanical, and volumetric laboratory test data. The dashboard provides interactive KPIs, trend analysis, material comparisons, and quality insights to support QA/QC monitoring.
