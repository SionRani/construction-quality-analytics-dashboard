# Construction Quality Analytics Dashboard

## Overview
This project presents an interactive Power BI dashboard developed to analyze construction quality using aggregate, mechanical, and volumetric laboratory test data. It provides interactive KPIs, trend analysis, material-wise comparisons, and quality insights to support QA/QC monitoring.

## Dashboard Pages
- Construction Quality Analytics Dashboard
- Quality Dashboard Overview
- Volumetric Test Analysis Dashboard

## Key Features
- Interactive KPI Cards
- Aggregate Quality Analysis
- Mechanical Test Analysis
- Volumetric Test Analysis
- Material-wise Comparison
- Trend Analysis
- Interactive Filters

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Microsoft Excel

## Project Inspiration
Inspired by QA/QC practices observed during my Larsen & Toubro (L&T) internship at the TIMS Hospital project.
